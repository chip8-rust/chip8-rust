fn drawer(&self) -> RenderDrawer

Provides drawing methods for the renderer.

Remarks

This method is not &mut self. It uses interior mutability via RenderDrawer<'renderer> to preserve the Renderer's lifetime, and therefore the lifetimes of any Textures that belong to the Renderer.

Only one RenderDrawer per Renderer can be active at a time. If this method is called and an existing RenderDrawer from this instance is active, the program will panic.

Examples

use sdl2::render::Renderer;
use sdl2::rect::Rect;

fn test_draw(renderer: &Renderer) {
    let mut drawer = renderer.drawer();
    drawer.clear();
    drawer.draw_rect(Rect::new(50, 50, 150, 175));
    drawer.present();
}
